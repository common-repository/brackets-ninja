== Installation ==

= Creating a bracket =

1. Create a FREE account at [Brackets Ninja](https://brackets.commoninja.com).
2. Create a new bracket by clicking the "Try it now" button on Brackets Ninja's homepage.
3. Customize the bracket. Make it beautiful. Add teams & players, select the bracket type (single elimination, double elimination, march madness, etc), and change the look and feel to match your brand.
4. Click the "Save Changes" button.

= Extracting the bracket ID =

1. Go to your account's [dashboard](https://user.commoninja.com/dashboard?type=bracket).
2. Find the bracket you want to add, and click the plugin's menu (3 dots icon).
3. Look for the "Add to Site" button and click it.
4. Under the embeddable code, you'll see a "Plugin ID" section. Copy the ID, and save it for later.

= Attaching a bracket to a Wordpress post =

1. Use Bracket Ninja's shortcode directly in the editor: [bracketsninja bracketid="YOUR_BRACKET_ID"].
2. Make sure you replace `YOUR_BRACKET_ID` with the one from "Extracting the bracket ID" section above.

Arguments provides by the shortcode:

* 'bracketid' - Your plugin ID from Common Ninja
